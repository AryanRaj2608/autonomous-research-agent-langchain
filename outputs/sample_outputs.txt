========================================
AUTONOMOUS RESEARCH AGENT - SAMPLE OUTPUT
========================================

This folder stores the research reports
generated by the Autonomous Research Agent.

Each report is saved with the format:
  research_report_<topic>.txt

Reports contain:
  - Title
  - Introduction
  - Key Insights
  - Applications
  - Advantages
  - Challenges
  - Conclusion

Run the agent to generate your first report:
  python autonomous_research_agent.py
